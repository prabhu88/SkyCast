# SkyCast
SkyCast : Your Personal Weather Assistant. SkyCast is a weather application built using Electron, Node.js, and React. The app allows users to get real-time weather data for their location or any other location they specify

## Features

- Real-time weather data for any location
- Current temperature, wind speed, precipitation, and other weather information
- Location-based forecasts
- Interactive maps

## Getting Started
To get started with SkyCast, follow these steps:
1. Clone the repository to your local machine using git clone <https://github.com/prabhu88/SkyCast.git>. 
2. Install Node.js and React on your machine.
3. Install Electron using npm install -g electron.
4. Navigate to the project directory and install the required dependencies using npm install.
5. Start the app by running npm start.

## Usage
When you open the app, it will automatically retrieve weather data for your current location. You can also search for weather data for other locations by entering a city or zip code in the search bar.

The app displays the current temperature, wind speed, precipitation, and other weather information. You can also view location-based forecasts and interactive maps.
